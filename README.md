# TaxScape Pro

R&D Tax Credit Suite - AI-powered tax credit analysis and documentation.

## Features
- AI Auditor chatbot for R&D qualification
- Excel report generation
- User authentication via Supabase
- Admin portal

## Tech Stack
- Frontend: Next.js, Tailwind CSS
- Backend: FastAPI, Python
- Database: Supabase (PostgreSQL)
- AI: Google Gemini
